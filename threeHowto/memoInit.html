Threeを使用する準備

<head>内に<script type="importmap">を記載してエイリアスを設定しておく。
<body>の最後に<script type="module">を記載して、実行するスクリプトを記述する。
<script type="module">の先頭には、必要なimport宣言を記述しておく。
<html> <head> ... <script type="importmap"> { "imports": { "three": "https://unpkg.com/three@0.164.0/build/three.module.js", "three/addons/": "https://unpkg.com/three@0.164.0/examples/jsm/" } } </script> </head> <body> ... <script type="module"> import * as THREE from 'three'; import { VRButton } from 'https://unpkg.com/three@0.164.0/examples/jsm/webxr/VRButton.js'; import { XRControllerModelFactory } from 'https://unpkg.com/three@0.164.0/examples/jsm/webxr/XRControllerModelFactory.js'; import { OrbitControls } from 'three/addons/controls/OrbitControls.js'; ... </script>

キャンバス(canvas)の設定

Threeは基本的に<canvas>内に描画する。<canvas>は<script>内で生成することもできるが、HTMLに定義しておく方が簡単。
とりあえず、ブラウザの画面全体を<canvas>とした例を記載する。
<html> <head> ... <style> #canvas { position: absolute; top: 0px; left: 0px; } </style> <script type="importmap"> </script> </head> <body> <div id="canvas" style="width: 100%; height: 100%;"></div> ... <script type="module"> ... </script>

Threeの基本要素

Threeでは、3Dオブジェクトが置かれた空間(scene)をカメラ(camera)から見た状態をレンダラ(renderer)に描画する。
また、通常のオブジェクトは発光しないので見えない。オブジェクトを見えるようにするために、この空間(scene)にはライト(light)も配置する。
ここでは、以下の順番で設定する。
  1. オブジェクトを空間(scene)に配置する。
  2. ライト(light)を空間(scene)に配置する。
  3. レンダラ(renderer)に空間(scene)とカメラ(camera)を設定する。
  4. キャンバス(canvas)にレンダラ(renderer)を追加する。
立方体を描画する例を記載する。
... const canvas = document.querySelector('#canvas'); function getWidth() { return canvas.getBoundingClientRect().width; } function getHeight() { return canvas.getBoundingClientRect().height; } // 空間(scene)を準備する。 const scene = new THREE.Scene(); // 描画対象の立方体を作って空間(scene)に配置する。 const geometry = new THREE.BoxGeometry(1, 1, 1); const material = new THREE.MeshStandardMaterial({color: 0xffff00, metalness: 0.5, roughness: 0.5}); const mesh = new THREE.Mesh(geometry, material); scene.add(mesh); // ライト(light)を作って空間(scene)に配置する。 const directionLight = new THREE.DirectionalLight(0xffffff, 3); directionLight.position.set(1, 1, 1); directionLight.target.position.set(0, 0, 0); scene.add(directionLight); const ambientLight = new THREE.AmbientLight(0x999999, 2); scene.add(ambientLight); // カメラ(camera)を作る。 const camera = new THREE.PerspectiveCamera(50, getWidth()/getHeight(), 0.1, 1000); camera.position.set(-1, 1, 2); camera.lookAt(0, 0, 0); // レンダラ(renderer)を作って、空間(scene)とカメラ(camera)を設定する。 const renderer = new THREE.WebGLRenderer(); renderer.setSize(getWidth(), getHeight()); renderer.setClearColor(0xdddddd); renderer.render(scene, camera); // キャンバス(canvas)にレンダラ(renderer)を追加する。 canvas.appendChild(renderer.domElement); ...

レンダラ(renderer)

通常は、WebGLRendererを使用する。以上。

カメラ(camera)

カメラは5種類あるが、通常使用するのは、PerspectiveCamera(透視投影カメラ)。用途によっては、遠近感のないOrthographicCamera(平行投影カメラ)を使うこともあるらしい。
new PerspectiveCamera( fov : number, aspect : number, near : number, far : number )
fovはField of View(視野角)の略でデフォルトは50。値が小さいと望遠レンズ、大きいと広角レンズのような表現になる。
aspectはアスペクト比で、横/縦。キャンバスのサイズから計算して設定する。
near, farは手前の描画限界と奥の描画限界。nearは0.1～1、farは 1000〜3000程度で設定するのが安全。

ライト(light)

6種類のライトがある。通常は、Directional(平行光) + LightAmbientLight(環境光)を使用する。
AmbientLight（環境光） シーン全体を均一に照らす。他のライトと併用するのが基本。補助光。
DirectionalLight（平行光） 太陽光のように 1方向から平行に照射。屋外シーンに最適。屋外。
HemisphereLight（半球光） 上（空）と下（地面）で色を変えられる。屋外の自然光に近い。屋外の雰囲気作り。
PointLight（点光源） 電球のように 一点から全方向へ照射。距離による減衰あり。屋内照明。
SpotLight（スポットライト） 懐中電灯のように 円錐状に照射。舞台・懐中電灯。
RectAreaLight（矩形ライト） 光る板のように 面から照射。窓・ライトパネルの表現に最適。
new AmbientLight( color : number | Color | string, intensity : number ) new DirectionalLight( color : number | Color | string, intensity : number ) new HemisphereLight( skyColor : number | Color | string, groundColor : number | Color | string, intensity : number ) new PointLight( color : number | Color | string, intensity : number, distance : number, decay : number ) new SpotLight( color : number | Color | string, intensity : number, distance : number, angle : number, penumbra : number, decay : number ) new RectAreaLight( color : number | Color | string, intensity : number, width : number, height : number )
colorは光の色、intensityは光の強さ(1 が標準、2〜5 で強めの光)。

おまけ：　カメラの移動

OrbitControlsを使って、マウスの操作でカメラを移動できるようにしておく。
const controls = new OrbitControls(camera, renderer.domElement); function animate() { controls.update(); renderer.setSize(getWidth(), getHeight()); renderer.render(scene, camera); } renderer.setAnimationLoop(animate);
