影(shadow)

概要

ここでは左にある影付きの立方体を描画するまでを解説する。（立方体の描画については承前）
影を付ける場合には、以下の設定が必要。
  • 光(light)に影(shadow)の設定をする。
  • 影を作るオブジェクトに影を落とす(caseShadow)設定をする。
  • 影を受けるオブジェクトに影を受ける(receiveShadow)設定をする。
  • レンダラ(renderer)に影の設定をする。
以下、立方体オブジェクトが大きな直方体に乗っている状態で、立方体オブジェクトの影を直方体オブジェクトに落とす例を示す。

光(light)に影(shadow)の設定をする。

const directionLight = new THREE.DirectionalLight(0xffffff, 3); directionLight.position.set(1, 1, 1); directionLight.target.position.set(0, 0, 0); directionLight.castShadow = true; directionLight.shadow.mapSize.width = 1024; directionLight.shadow.mapSize.height = 1024; directionLight.shadow.camera.left = -5; directionLight.shadow.camera.right = 5; directionLight.shadow.camera.top = 5; directionLight.shadow.camera.bottom = -5; directionLight.shadow.camera.near = 0.1; directionLight.shadow.camera.far = 100; directionLight.shadow.bias = -0.0001; scene.add(directionLight);

影を作るオブジェクトに影を落とす(caseShadow)設定をする。

const geometry = new THREE.BoxGeometry(1, 1, 1); const material = new THREE.MeshStandardMaterial({color: 0xffff00, metalness: 0.5, roughness: 0.5}); const mesh = new THREE.Mesh(geometry, material); mesh.castShadow = true; scene.add(mesh);

影を受けるオブジェクトに影を受ける(receiveShadow)設定をする。

const geometry2 = new THREE.BoxGeometry(5, 1, 5); const material2 = new THREE.MeshStandardMaterial({color: 0xa0a0a0}); const field = new THREE.Mesh(geometry2, material2); field.position.y = -1; field.receiveShadow = true; scene.add(field);

レンダラ(renderer)に影の設定をする。

const renderer = new THREE.WebGLRenderer(); renderer.setSize(getWidth(), getHeight()); renderer.setClearColor(0xdddddd); renderer.shadowMap.enabled = true; renderer.render(scene, camera);
